## 📋 Список скриптов

### 📥 Скачивание видео с YouTube

**Файл:** `dowload_videos_from_youtube.py`

Простой скрипт для скачивания видео с YouTube в максимальном качестве.

**Как работает:**
- Принимает URL видео
- Скачивает в папку `Downloads`
- Использует библиотеку `pytube`

**Использование:**
```python
# Изменить URL в скрипте
URL = 'https://youtube.com/your_video_link'

# Запустить
python download_videos_from_youtube.py
```
**Зависимости**
```bash
pip install pytube
```

### 📝 Генерация логов изменений

**Файл:** `log_generation.py`

Скрипт отслеживает изменения файлов в указанной директории и записывает их в лог-файл.

**Как работает:**
- Сканирует все файлы в директории (рекурсивно)
- Проверяет время последнего изменения (mtime, atime, ctime)
- Если файл изменён за последние 87 000 секунд (~24 часа) — записывает в лог

**Использование:**
```bash
# Запуск с указанием директории
python log_generation.py D:\MyFolder

# Запуск с запросом пути (интерактивный режим)
python log_generation.py
```
**Настройки:**
```pyhon
TIME_BORDER = 87000          # Время для отслеживания (секунды)
CHECK_DIRECTORY = r'D:\PythonProjects'  # Путь по умолчанию
FILE_LOG = r'D:\PythonProjects\viruses\log.txt'  # Путь к логу
```

### 📂 Переименование файлов

**Файл:** `rename_files.py`

Массовое переименование файлов по заданному правилу.

**Как работает:**
- Проходит по всем файлам в директории (рекурсивно)
- Добавляет префикс A_ к имени файла, если его нет
- Заменяет _Diff. на _BC

**Использование:**
```pyhon
# Изменить путь в скрипте
DIRECTORY = r'D:\PythonProjects'

# Запустить
python rename_files.py
```
**Пример:**
```text
До: image_Diff.jpg
После: A_image_BC.jpg
```
### 🎤 Конвертация WAV в текст

**Файл:** `conv_wav_to_txt.py`

Распознавание речи из аудиофайла (WAV) с помощью Google Speech Recognition.

**Как работает:**
- Загружает WAV-файл
- Отправляет на распознавание в Google Speech Recognition
- Возвращает распознанный текст

**Использование:**
```pyhon
# Изменить путь к файлу в скрипте
FILE_PATH = 'audio.wav'
LANGUAGE = 'ru-RU'  # Язык распознавания (ru-RU, en-EN)

# Запустить
python conv_wav_to_txt.py
```

**Зависимости:**
```bash
pip install SpeechRecognition pydub
```

**Сохранение результата в файл:**
```pyhon
# Раскомментировать в конце скрипта
with open('result.txt', 'w', encoding='utf-8') as f:
    f.write(result)
```

### 🖼️ Конвертация изображений

**Файл:** `convert_images.py`

Массовая конвертация изображений: смена формата, сжатие до заданного размера.

**Как работает:**
- Проходит по всем файлам в директории (рекурсивно)
- Находит файлы с расширением .jpg
- Конвертирует в .png
- Сжимает до размера 1024x1024 (сохраняя пропорции)
- Удаляет исходный JPG-файл

**Использование:**
```pyhon
# Изменить настройки в скрипте
DIRECTORY = r'D:\PythonProjects\test'  # Папка с изображениями
FROM_EXTENTION = '.jpg'                # Исходный формат
TO_EXTENTION = '.png'                  # Целевой формат
MAX_SIZE = (1024, 1024)                # Максимальный размер

# Запустить
python convert_images.py
```

**Зависимости:**
```bash
pip install Pillow
```

**Сохранение результата в файл:**
```pyhon
# Раскомментировать в конце скрипта
with open('result.txt', 'w', encoding='utf-8') as f:
    f.write(result)
```
